# Todo
Todo Application Spring Boot
Getting started:
    1. Clone the repository.
    2. mvn clean install.
    3. cd ./target
    4. java -DPORT=your_port -jar todo-0.0.1-SNAPSHOT.jar
   
   
Api documentation: 
http://host:port/todo/swagger-ui.html (UI) 
or 
http://host:port/todo/v2/api-docs (JSON)
